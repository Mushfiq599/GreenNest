GreenNest is a modern, responsive web application for discovering indoor plants, exploring plant categories, and getting personalized plant care guidance.

## Features

###  Authentication
- Email & Password Login
- Google Social Login
- User Registration with Profile Setup
- Forgot Password (Password reset via email)
- Protected Routes (Only Home is public)

### Plant Browsing
- Category-based plant listing
- Detailed plant information
- Next / Previous navigation between plants
- Stock, rating, price, provider info

### Book Consultation
- Book plant care consultation
- Form with name & email
- Success notification on submission

###  User Profile
- View profile (Name, Email, Photo)
- Update display name & photo using Firebase

###  UI & UX
- Fully responsive (mobile, tablet, desktop)
- Tailwind CSS + DaisyUI
- Toast notifications for success & errors
- Custom 404 / Not Found page
